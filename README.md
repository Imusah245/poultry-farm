# Poultry Farm Website 🐔

A modern poultry farm website built with Vite and React.

![Logo](src/assets/logo.png)

## Features
- Egg production section
- Broiler production section
- Farm information
- Contact page
- Responsive UI
- Real-time dashboard (mock data)
- Analytics charts
- Admin login
- Scroll reveal animations

## Tech Stack
- React
- Vite
- Tailwind CSS
- Chart.js
- Framer Motion
